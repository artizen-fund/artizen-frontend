# Artizen Frontend

- NextJS 👷
- Typescript 🧑‍💻
- styled-components 💅🏻
- Cypress testing 🧑‍🏫
- Component Storybooks 🖼

Coming soon…

- Jest testing 🤪
- react-jsonschema-forms 📝

## Getting Started

Run the development server:

```bash
yarn dev
```

Run storybooks:

```bash
yarn storybook
```

Run Cypress tests:

- make sure sure dev server is running, then…

```bash
yarn test
```
