Featuring…

- NextJS 👷
- Typescript 🧑‍💻
- styled-components 💅🏻
- Cypress testing 🧑‍🏫
- Component Storybooks 🖼

Coming soon…

- Jest testing 🤪

## Installation

- requires Node 16.10.0+
- npx storybook init

## Getting Started

### Run the development server:

```bash
yarn dev
```

### Create a new component:

```bash
yarn generate
```

### Run storybooks:

```bash
yarn storybook
```

### Run Cypress tests:

- make sure sure dev server is running, then…

```bash
yarn test
```
