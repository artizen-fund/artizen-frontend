# Artizen Frontend

Artizen.fund codebase

Featuring…

- NextJS 👷
- Typescript 🧑‍💻
- styled-components 💅🏻
- Cypress testing 🧑‍🏫
- Jest testing 🤪
- Component Storybooks 🖼
- react-jsonschema-forms 📝

## Installation

- requires Node 16.14.2+
- npx storybook init

## Getting Started

### Run the development server:

```bash
yarn dev
```

### Graphql codegen:

```bash
yarn codegen
```

### Create a new component:

```bash
yarn generate
```

### Run storybooks:

```bash
yarn storybook
```

### Run Cypress tests:

- make sure sure dev server is running, then…

```bash
yarn test
```

## Run Jest tests:

```bash
yarn test:jest
```
